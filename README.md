# Code Garden

Pythonの基本をダンジョン攻略として体験する、30分の体験研修用プロトタイプです。

## 学習順

0. 操作チュートリアル
1. `move()`と`action()`
2. `print()`と文字列
3. `input()`と変数
4. `for`
5. 変数と`if / else`

## Googleスプレッドシートへの記録

開始時に氏名、開始時刻、セッションIDを作成します。各階層をクリアすると、階層開始時刻、クリア時刻、所要秒数、クリア時のソースコードを送信します。

1. Googleスプレッドシートを新規作成する。
2. 「拡張機能」→「Apps Script」を開く。
3. リポジトリの `google-apps-script.gs` を貼り付けて保存する。
4. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」を選択する。
5. 実行ユーザーを自分、アクセスできるユーザーを研修環境に合わせて設定してデプロイする。
6. 発行されたウェブアプリURLを `config.js` に設定する。

```js
window.CODE_GARDEN_RECORD_ENDPOINT = 'ここにウェブアプリURL';
```

URL未設定または通信失敗時の記録は、ブラウザーの `code-garden-record-queue` に保留されます。次回送信時に再試行されます。

記録することを開始画面で本人へ明示し、氏名や提出コードの保存期間・閲覧権限は研修所の個人情報管理ルールに合わせてください。
